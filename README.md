# 纸张工具(ZhiZhangConee)功能介绍

## 项目概述
这是一个基于 `ncatbot` 框架开发的多功能QQ机器人插件，提供丰富的娱乐和实用功能。

## 功能列表

### 🤖 基础功能
- **自动回复**: 智能识别关键词并自动回复
- **帮助系统**: 输入"智障帮助"查看所有可用功能

### 🎮 娱乐互动
- **Kepp游戏**: 请说`kepp帮助`查看详细玩法
- **猜拳游戏**: 输入`猜拳帮助`开始游戏
- **猜数字**: 输入`猜数字帮助`参与数字猜测游戏
- **复读功能**: 使用`复读:+内容`进行复读

### 💰 模拟功能
- **模拟存钱**: 输入`存钱帮助`了解详情
- **打我功能**: 输入`打我`体验互动
- **赞我功能**: 输入`赞我`获得点赞

### 🌤️ 实用工具
- **天气查询**: 输入`天气帮助`获取天气信息
- **八字测算**: 输入`八字帮助`进行生辰八字分析
- **AI聊天**: 输入`ai聊天帮助`开启智能对话

### 🖼️ 图片处理
- **颜色反转**: 引用图片并@机器人+图片反转
- **图片对称**: 回复图片并使用上/下/左/右对称处理
- **表情合成**: @机器人+`表情合成:合成类型`制作表情包

### 🖥️ 系统控制
- **电脑关机**: 使用`电脑关机`/`取消关机`远程控制
- **随机图片**: 发送`#看看美女`、`#看看帅哥`等指令获取图片

### 👥 群管功能
- **禁言管理**: 输入`禁言帮助`查看管理员功能
- **机器人管理**: 输入`管理帮助`进行机器人设置

### 🐍 开发者工具
- **Python代码运行**: 输入`运行帮助`在线执行Python代码

## 使用说明
1大部分功能都需要通过特定关键词触发，建议先输入"智障帮助"查看完整功能列表和使用方法。

2下载解压后，把目录里面的ZhiZhangConee放入plugins文件

3表情包合成 功能请运行根目录下的txbqb\_api.py
